# Changelog

Todas as alterações notáveis neste projeto serão documentadas neste arquivo.

O formato segue [Keep a Changelog](https://keepachangelog.com/pt-BR/1.1.0/),
e este projeto adere ao [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2026-06-02

### Adicionado
- Estrutura MVC própria em PHP 8.3+
- Sistema de rotas com suporte a parâmetros dinâmicos e middlewares
- Autenticação unificada com identificação automática de perfil
- Sistema de permissões por role (Super Admin, Sócio, Desenvolvedor, Suporte, Comercial, Financeiro, Cliente)
- Dashboard administrativo com cards, gráficos e timeline
- Módulo de gerenciamento de clientes (CRM completo)
- CRM comercial com funil Kanban
- Sistema de orçamentos com blocos, cálculos automáticos e link público
- Módulo de projetos com Kanban e tarefas
- Módulo financeiro (receitas, despesas, fluxo de caixa)
- Gestão documental com categorias e permissões
- Gestão de equipe com perfis e permissões configuráveis
- Área do cliente com dashboard exclusivo
- Site institucional multilíngue (PT, EN, ES)
- Sistema de portfólio dinâmico com filtros
- Blog com integração OpenAI (geração automática)
- CMS de páginas editáveis
- SEO avançado (Open Graph, Twitter Cards, Schema.org, Sitemap, Robots)
- Dark Mode em todo o sistema
- Banner de cookies (LGPD)
- Sistema de migrations para banco de dados
- Sistema de logs e auditoria
- Módulo de versionamento interno
- Módulo de backups
- Proteção CSRF, XSS, SQL Injection
- Rate Limiting
- Cron Job para blog com IA
- Preparação para CI/CD futuro
- API REST v1 para expansão
